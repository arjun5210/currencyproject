Managing cache
To prevent unnecessary API calls in this project, I implemented a caching mechanism using a Map to store API responses. When a user makes a request with the same parameters repeatedly within a 5-minute window, the system first checks if the response already exists in the cache. If found, the data is retrieved from the cache instead of making a new API call. This approach significantly reduces redundant network requests and improves the overall performance of the application.
